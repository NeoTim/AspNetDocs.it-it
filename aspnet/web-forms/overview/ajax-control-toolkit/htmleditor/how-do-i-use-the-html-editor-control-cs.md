## <a name="avoiding-cross-site-scripting-xss-attacks"></a><span data-ttu-id="edde0-152">Evitare attacchi di cross-site scripting (XSS)</span><span class="sxs-lookup"><span data-stu-id="edde0-152">Avoiding Cross-Site Scripting (XSS) Attacks</span></span>

<span data-ttu-id="edde0-153">Ogni volta che si accetta l'input di un utente e si rivisualizza tale input nel sito Web, è possibile aprire il sito Web agli attacchi XSS (Cross-Site Scripting).</span><span class="sxs-lookup"><span data-stu-id="edde0-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="edde0-154">In teoria, un hacker malintenzionato potrebbe inviare codice JavaScript che viene eseguito quando l'input viene visualizzato nuovamente.</span><span class="sxs-lookup"><span data-stu-id="edde0-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="edde0-155">Il JavaScript potrebbe essere utilizzato per rubare le password degli utenti o altre informazioni sensibili.</span><span class="sxs-lookup"><span data-stu-id="edde0-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>

<span data-ttu-id="edde0-156">Normalmente, è possibile sconfiggere gli attacchi XSS con la codifica HTML qualsiasi input recuperato da un utente prima di visualizzarlo in una pagina web.</span><span class="sxs-lookup"><span data-stu-id="edde0-156">Normally, you can defeat XSS attacks by HTML encoding whatever input you retrieve from a user before displaying it in a web page.</span></span> <span data-ttu-id="edde0-157">Tuttavia, la codifica HTML l'output dell'editor HTML non solo codificai &lt;i tag di script,&gt; ma codifica va anche tutti i tag HTML.</span><span class="sxs-lookup"><span data-stu-id="edde0-157">However, HTML encoding the output of the HTML Editor would not only encode &lt;script&gt; tags, it would also encode all HTML tags.</span></span> <span data-ttu-id="edde0-158">In altre parole, si perderebbe tutta la formattazione, ad esempio il tipo di carattere, la dimensione del carattere e il colore di sfondo.</span><span class="sxs-lookup"><span data-stu-id="edde0-158">In other words, you would lose all of the formatting such as the font type, font size, and background color.</span></span>
