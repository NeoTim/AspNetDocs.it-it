# <a name="multi-targeting-support-in-visual-studio-2008"></a>Supporto multitargeting in Visual Studio 2008

di [Joe Stagner spiega](https://github.com/JoeStagner)

Illustrerò la funzionalità multitarget di Visual Studio 2008. In passato era necessario scegliere la versione di Visual Studio in modo che corrisponda alla versione .NET di destinazione. Ad esempio, Visual Studio 2005 ha funzionato con .NET 2,5. Con Visual Studio 2008 è ora possibile scegliere la versione di .NET appropriata per l'applicazione.
