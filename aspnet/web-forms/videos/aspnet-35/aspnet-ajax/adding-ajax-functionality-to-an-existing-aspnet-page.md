# <a name="adding-ajax-functionality-to-an-existing-aspnet-page"></a>Aggiunta di funzionalità AJAX a una pagina ASP.NET esistente

di [Todd Miranda](https://github.com/tmiranda)

L'uso di AJAX nelle pagine di ASP.NET può migliorare le prestazioni e migliorare l'esperienza utente complessiva di un'applicazione. Non è necessario riscrivere le pagine esistenti per aggiungere funzionalità AJAX. Con solo poche modifiche, è possibile eseguire una pagina ASP.NET esistente per l'uso di AJAX. In questo video viene illustrato come aggiungere la funzionalità AJAX a una pagina ASP.NET esistente con il minimo sforzo.
