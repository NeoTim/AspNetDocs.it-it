# <a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[Procedura:] Usare il controllo ASP.NET AJAX UpdateProgress?

di [Joe Stagner spiega](https://github.com/JoeStagner)

In un'applicazione AJAX è spesso presente una pausa notevole tra il momento in cui il browser esegue una richiesta di rete e il momento in cui il browser riceve una risposta. Per informare l'utente che è in corso la conversazione di rete, il controllo ASP.NET AJAX UpdateProgress fornisce un feedback immediato visualizzando un'immagine animata, il testo di caricamento o altro codice HTML che scompare dopo la ricezione della risposta di rete.
