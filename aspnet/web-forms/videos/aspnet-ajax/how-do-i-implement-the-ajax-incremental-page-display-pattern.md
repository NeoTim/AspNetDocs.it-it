# <a name="how-do-i-implement-the-ajax-incremental-page-display-pattern"></a>[Procedura:] Implementare il modello di visualizzazione incrementale delle pagine AJAX?

di [Joe Stagner spiega](https://github.com/JoeStagner)

Tradizionalmente, quando viene richiesta una pagina Web, l'utente deve attendere che l'intera pagina venga recuperata dal server dopo la quale la pagina completa viene visualizzata improvvisamente nel browser. In questo video viene illustrato il modo in cui il modello di visualizzazione incrementale delle pagine AJAX migliora l'esperienza utente con il recupero e la visualizzazione di parti della pagina Web in modo rapido e semplice.
