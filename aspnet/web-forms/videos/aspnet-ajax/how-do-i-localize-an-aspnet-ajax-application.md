# <a name="how-do-i-localize-an-aspnet-ajax-application"></a>[Procedura:] Localizzare un'applicazione ASP.NET AJAX?

di [Chris Pels](https://twitter.com/chrispels)

In questo video si inizia con una panoramica delle funzionalità di localizzazione del framework ASP.NET standard e quindi si esaminano le funzionalità di localizzazione di ASP.NET AJAX Extensions. Viene illustrato come è possibile localizzare le informazioni relative a testo, data e numero nel client in base alle impostazioni della lingua del browser. Viene inoltre illustrato il modo in cui l'applicazione AJAX può utilizzare di nuovo le risorse statiche e incorporate nel server.
