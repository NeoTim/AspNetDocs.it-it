# <a name="how-do-i-customize-error-handling-for-the-aspnet-ajax-updatepanel"></a>Ricerca per categorie: personalizzazione della gestione degli errori per ASP.NET AJAX UpdatePanel

di [Rob Windsor](https://twitter.com/robwindsor)

Quando si verifica un errore durante l'elaborazione di un postback attivato da un UpdatePanel ASP.NET AJAX, il PageRequestManager sul lato client visualizzerà un avviso (.NET 2,0 o 3,0) o semplicemente rigenererà l'eccezione (.NET 3,5 o versione successiva). In entrambi i casi, l'esperienza per l'utente finale è meno corretta. In questo video Rob Windsor dimostrerà come risolvere questo problema personalizzando la gestione degli errori associati a UpdatePanel
