# <a name="how-do-i-use-the-aspnet-ajax-nobot-control"></a>[Procedura:] Usare il controllo NoBot ASP.NET AJAX?

da [Joe Stagner](https://github.com/JoeStagner)

Controllo NoBot di ASP.NET AJAX Control Toolkit consente di evitare alcuni tipi di accesso non autorizzato alle applicazioni Web. In questo video viene illustrato come il controllo NoBot fornisce una serie di incorporati protegge contro l'accesso automatico, tra cui verificare il numero e la frequenza di invii di form. Noteremo anche come il controllo consente allo sviluppatore di definire una sfida personalizzata, ad esempio un test di JavaScript che verrÃ  completata solo se la pagina viene visualizzata in un browser.
