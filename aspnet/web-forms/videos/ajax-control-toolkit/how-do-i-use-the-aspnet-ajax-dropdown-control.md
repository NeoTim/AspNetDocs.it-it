# <a name="how-do-i-use-the-aspnet-ajax-dropdown-control"></a>[Procedura:] Usare il controllo DropDown ASP.NET AJAX?

di [Joe Stagner spiega](https://github.com/JoeStagner)

Con il dispositivo Extender di controllo a discesa di ASP.NET AJAX Control Toolkit, è possibile presentare all'utente un elenco a discesa di opzioni che attivano un aggiornamento della pagina parziale in un'altra posizione nella pagina Web. L'opzione selezionata dall'utente viene passata al server, consentendo al codice sul lato server di elaborare la selezione prima di completare l'aggiornamento della pagina parziale.
