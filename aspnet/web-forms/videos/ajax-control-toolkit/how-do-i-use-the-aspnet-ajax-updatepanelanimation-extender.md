# <a name="how-do-i-use-the-aspnet-ajax-updatepanelanimation-extender"></a>[Procedura:] Usare il dispositivo Extender ASP.NET AJAX UpdatePanelAnimation?

di [Joe Stagner spiega](https://github.com/JoeStagner)

Aggiungendo un'animazione a un aggiornamento a pagina parziale è possibile informare l'utente quando si verifica un aggiornamento e al termine. Il dispositivo Extender UpdatePanelAnimation aggiunge gli effetti di animazione al controllo UpdatePanel ASP.NET AJAX, usando il Framework di animazione completo delle estensioni ASP.NET AJAX di base. In questo video viene illustrato come è possibile definire la sequenza di effetti per UpdatePanelAnimation e come è possibile ottimizzare l'animazione utilizzando le definizioni di ScriptAction.
