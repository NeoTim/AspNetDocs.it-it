# <a name="how-do-i-use-the-aspnet-ajax-tabs-control"></a>[Procedura:] Usare il controllo ASP.NET AJAX Tabs?

di [Joe Stagner spiega](https://github.com/JoeStagner)

Questo video illustra le nozioni di base del controllo schede di ASP.NET AJAX Control Toolkit, con cui è possibile creare un set di schede per organizzare il contenuto in una pagina Web. La scheda selezionata viene mantenuta tra i postback, rendendo le schede il controllo ideale per le pagine di amministrazione basata su form.
