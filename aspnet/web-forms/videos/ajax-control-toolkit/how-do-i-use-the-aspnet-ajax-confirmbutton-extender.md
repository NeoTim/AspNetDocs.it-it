# <a name="how-do-i-use-the-aspnet-ajax-confirmbutton-extender"></a>[Procedura:] Usare il dispositivo Extender ConfirmButton ASP.NET AJAX?

da [Joe Stagner](https://github.com/JoeStagner)

Scopri come il dispositivo extender ConfirmButton di ASP.NET AJAX Control Toolkit fornisce un modo semplice per aggiungere una finestra di dialogo di conferma per l'evento click di un controllo pulsante, LinkButton e ImageButton. Ciò fornisce all'utente la possibilità di annullare un'azione imminente.
