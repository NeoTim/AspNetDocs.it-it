# <a name="how-do-i-use-the-numericupdown-extender-control"></a>[Procedura:] Usare il controllo Extender NumericUpDown?

di [Joe Stagner spiega](https://github.com/JoeStagner)

In questo video viene esaminato il controllo Extender NumericUpDown da ASP.NET AJAX Control Toolkit. Viene illustrato il modo in cui il dispositivo Extender NumericUpDown aggiunge i pulsanti "up" e "Down" a un controllo TextBox ASP.NET standard. I pulsanti consentono all'utente di incrementare o decrementare un numero, scorrere un elenco di valori noti o chiamare un servizio Web per recuperare il valore successivo. In questo modo lo sviluppatore può controllare completamente i valori che l'utente può inviare a un'applicazione Web.
