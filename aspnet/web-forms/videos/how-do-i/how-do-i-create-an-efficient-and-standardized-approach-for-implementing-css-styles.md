# <a name="how-do-i-create-an-efficient-and-standardized-approach-for-implementing-css-styles"></a>Ricerca per categorie: creazione di un approccio efficiente e standardizzato per l'implementazione di stili CSS

di [Chris Pels](https://twitter.com/chrispels)

In questo video Chris Pels illustra come organizzare e implementare stili CSS che forniscono un aspetto standardizzato a un sito Web. Gli stili sono inoltre organizzati per offrire la massima flessibilità e riusabilità. Viene creato un set di stili che è incentrato sull'aspetto visivo del sito. Per prima cosa, viene creato un sito Web di esempio con un file CSS per una pagina di esempio. I primi gruppi di stili creati sono orientati alla fornitura di contenitori per il contenuto in tutto il sito. Viene quindi aggiunto un set di stili che si concentrano sulla visualizzazione dello "stato" degli elementi dell'interfaccia utente. Infine, viene creato un set di stili per i segnali di interazione utente. Ognuno degli stili viene utilizzato con gli elementi HTML nella pagina di esempio. È inoltre disponibile una discussione sull'utilizzo dello stesso approccio di implementazione per gli stili che controllano il layout degli elementi nel sito.
