# <a name="how-do-i-add-controls-to-an-aspnet-web-page-programmatically"></a>[Procedura:] Aggiungere controlli a una pagina Web ASP.NET a livello di codice

di [Chris Pels](https://twitter.com/chrispels)

In questo video Chris Pels mostrerà come creare un controllo server ASP.NET a livello di codice in fase di esecuzione e aggiungerlo a una pagina. In primo luogo, viene illustrato il concetto di base di aggiunta di un controllo a una pagina a livello di codice. Viene quindi esaminato il concetto di controllo "container", che è necessario per l'aggiunta di un nuovo controllo a livello di codice in fase di esecuzione. Si apprenderanno quindi alcune considerazioni relative all'annidamento dei controlli aggiunti a livello di codice e alla creazione di metodi distinti che possono compartimenti il processo per renderlo più gestibile. Inoltre, vengono fornite le considerazioni per la gestione dello stato di visualizzazione quando si aggiungono controlli in modo dinamico.
