# <a name="how-do-i-tune-web-application-performance-with-profiling"></a>[Procedura:] Ottimizzare le prestazioni delle applicazioni Web con la profilatura?

di [Chris Menegay](https://twitter.com/CMenegay)

In questo video viene illustrato che Visual Studio Team System fornisce due metodi per la profilatura di un'applicazione Web: campionamento e strumentazione. La strumentazione inserisce i marcatori nel codice sottostante in modo che lo sviluppatore possa visualizzare i dettagli completi dell'esecuzione dell'applicazione, inclusi gli alberi delle chiamate e l'allocazione degli oggetti, per ottimizzare le prestazioni dell'applicazione.
