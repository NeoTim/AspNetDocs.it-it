# <a name="jquery-cycle-294"></a>jQuery Cycle 2.94

I file di ciclo jQuery seguenti sono ospitati in questa rete CDN:

- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.js) &mdash; la versione completa del plug-in del ciclo jQuery. Include tutti i più di due dozzine di effetti speciali e tutte le opzioni.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.min.js) &mdash; la versione minimizzati del file precedente. Usare questa versione per le applicazioni di produzione per migliorare le prestazioni del sito Web.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.js) &mdash; questa versione del plug-in del ciclo jQuery è più piccola e contiene solo l'effetto di transizione "fade".
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.min.js) &mdash; la versione minimizzati del file precedente. Usare questa versione per le applicazioni di produzione per migliorare le prestazioni del sito Web.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.js) &mdash; una versione ancora più piccola del plug-in del ciclo jQuery che contiene solo la transizione "dissolvenza" senza tutte le opzioni.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.min.js) &mdash; la versione minimizzati del file precedente. Usare questa versione per le applicazioni di produzione per migliorare le prestazioni del sito Web.
