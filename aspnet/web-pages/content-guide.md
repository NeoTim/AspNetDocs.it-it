# <a name="webmatrix-content-guide"></a>Guida ai contenuti su WebMatrix

> [!NOTE] 
> WebMatrix non è più consigliato come Integrated Development Environment per Pagine Web ASP.NET. Usare [Visual Studio](xref:aspnet/web-pages/overview/getting-started/program-asp-net-web-pages-in-visual-studio) o [Visual Studio Code](https://code.visualstudio.com/).

WebMatrix è un set di strumenti che consente di creare in modo semplice siti Web per Microsoft Windows. Include IIS Express (un server Web di sviluppo), ASP.NET (un Framework di programmazione Web) e SQL Server Compact (un database incorporato). Per una presentazione rapida di WebMatrix, vedere [Panoramica di WebMatrix](https://www.microsoft.com/web/webmatrix/). Per esaminare le principali funzionalità di WebMatrix, vedere [funzionalità](https://www.microsoft.com/web/webmatrix/features/)di WebMatrix.
