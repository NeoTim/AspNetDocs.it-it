## <a name="circuit-breakers"></a><span data-ttu-id="39dab-134">Interruttori</span><span class="sxs-lookup"><span data-stu-id="39dab-134">Circuit breakers</span></span>

<span data-ttu-id="39dab-135">Esistono diversi motivi per cui non si vuole ritentare troppe volte per un periodo troppo lungo:</span><span class="sxs-lookup"><span data-stu-id="39dab-135">There are several reasons why you don't want to retry too many times over too long a period:</span></span>
