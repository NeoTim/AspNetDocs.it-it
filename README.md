# <a name="aspnet-docs"></a>ASP.NET Docs

Questo repository contiene ospitata sul sito ASP.NET 4.x documentazione concettuale [docs.microsoft.com/aspnet](https://docs.microsoft.com/aspnet). Vedere le [guida per i contributi](CONTRIBUTING.md) e l'[l'elenco dei problemi](https://github.com/aspnet/AspNetDocs/issues) se si vuole contribuire.

Le modifiche alla documentazione dell'API devono essere apportate nel [repository ApiDocs](https://github.com/aspnet/ApiDocs) per i commenti con tripla barra `///`.
